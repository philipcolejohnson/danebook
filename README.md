This site shows the faces of Danes and puts them in a book-like format.

-a Ruby on Rails RESTful web app

-MVC architecture

-PostgreSQL backend

-Photo storage on Amazon S3

It is a project for [Viking Code School](http://vikingcodeschool.com)
